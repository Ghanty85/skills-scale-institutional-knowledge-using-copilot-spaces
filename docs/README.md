# OctoAcme Project Management Docs

## Overview
OctoAcme uses a customer-focused, iterative delivery approach grounded in clear roles, artifact-based planning, and transparent communication. Core project management processes include:
- Project initiation and one-pager chartering
- Iterative planning and prioritization
- Execution with tracking and quality safeguards
- Risk communication and escalation
- Structured retrospectives for continuous improvement

This documentation provides guidance and templates for each phase of the OctoAcme project lifecycle, helping teams align, deliver consistently, and quickly onboard new members.
